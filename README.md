# 🎯 SEO Article Generator Pro

Автоматическая генерация коммерческих статей для закупки ссылок. Работает полностью бесплатно.

## 🚀 Быстрый старт
1. Получите бесплатный API ключ: https://console.groq.com/keys
2. Вставьте ключ в `.streamlit/secrets.toml`
3. Запустите: `streamlit run app.py`
4. Откройте: http://localhost:8501

## 🌐 Деплой на Streamlit Cloud (бесплатно)
